What My Project Does
I made this Water Drop Game which is a game where the player controls a jerry can and tries to catch clean water drops while avoiding dirt balls. The game includes score tracking, lives, combo boosts, difficulty selection, stamina, dash movement, special pickups, and event waves that change the pacing of play which I recently added.

Technologies Used In This Project
- HTML
- CSS
- JavaScript

What I Learned
- How to handle keyboard and touch controls together
- How to manage game states like start, countdown, playing, pause, and game over
- How to tune gameplay with difficulty settings and timed events

Challenges Solved
- It was difficult making the game work on both desktop and mobile devices, but I got it working
- It was difficult keeping movement, spawning, and collision detection responsive at first
- Balancing helpful items, hazards, and scoring so the game feels fair enough
- Adding special events without breaking the main gameplay loop
- Resetting the game cleanly when restarting or returning home which was glitching at first
